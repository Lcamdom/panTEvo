# panTEvo
Here you will find the scripts used for cotton __LTR-Retrotransposon and solo-LTR annnotation__, the development of _G. hirsutum_ and _G. barbadense_ __pangenomes__, __TIP (Transposon Insertion Polymorphism)__ annotation, expression analyses (__TIP-eQTLs__) and cotton __pangenome genotyping__.\
\
This research is now published in Genome Biology: \
Campos-Dominguez, L., Castanera, R., Grover, C.E. et al. Differential LTR-retrotransposon dynamics across polyploidization, speciation, domestication, and improvement of cotton (Gossypium). Genome Biol 26, 369 (2025). [https://doi.org/10.1186/s13059-025-03837-7](https://doi.org/10.1186/s13059-025-03837-7).
